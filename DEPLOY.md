# Deploy for free

This app is a Streamlit quiz app. The easiest free public hosting option is
Streamlit Community Cloud.

## Streamlit Community Cloud

1. Fork or upload this project to your GitHub account.
2. Go to Streamlit Community Cloud and sign in with GitHub.
3. Create a new app from your repository.
4. Set the main file path to `agile.py`.
5. Deploy.

After deployment, Streamlit gives you a public URL like:

```text
https://your-app-name.streamlit.app
```

Keep these files in the repo:

- `agile.py`
- `agile.json`
- `requirements.txt`
- `.streamlit/config.toml`
